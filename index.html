Quarantine

Technology:

Platform:

Status:

Quarantine is a First Person Shooter game with elements of stealth and horror. I made this short game to learn Unity by creating a full game for the first time.

The game was inspired by games like Naughty Dog's Last Of Us, Bethesda's Dishonored and Ubisoft's Far Cry series.

  • The AI was improvised due to the lack of animations for the zombie models.
  • Majority of the assets were downloaded for free from the Unity asset store and other external sources.
  • Sound effects were modified using Audacity.

Cosmic Drift is a 2D Side-scrolling shooter game inspired by the gameplay of both Jetpack Joyride and Space Impact.

The game features the classic space shooter combat system similar to Space Impact while utilizing an endless runner concept similar to Jetpack Joyride.

  • Sprites and graphics used were free and modified accordingly.
